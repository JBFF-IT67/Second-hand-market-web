1.ใช้ PHP + MySQL PDO
2.มี ระบบ Login/Logout และแบ่งสิทธิ์อย่างน้อย 2 บทบาท: Admin / User
3.มี CRUD ครบ: เพิ่ม/แก้ไข/ลบ/ค้นหา
4.มีการใช้งาน รูปภาพอย่างน้อย 1 จุด (เช่น รูปสินค้า/ปก/โปสเตอร์/โปรไฟล์) พร้อมอัปโหลดไฟล์
5.มีหน้า Dashboard (Backend) สรุปข้อมูลอย่างน้อย 3 ตัวชี้วัด (เช่น จำนวนรายการ, จำนวนผู้ใช้, รายการล่าสุด)
6.มีการตรวจสอบข้อมูล (Validation) + , จำกัดชนิดไฟล์รูป (ปรับขนาดรูป เปลี่ยนชื่อรูป)
7.มีเอกสารส่ง: Data Dictionary + สรุปการแบ่งงาน 4 คน

Second-Hand Market: ระบบประกาศขายของมือสอง
ขอบเขตงาน: ประกาศขาย, หมวดหมู่, รูปสินค้า (หลายรูปได้), สถานะขาย
Frontend: ฟีดประกาศ, รายละเอียดประกาศ+รูป, ค้นหา/กรองตามหมวด/ราคา, หน้าประกาศของฉัน
Backend: จัดการหมวดหมู่, ตรวจประกาศผิดกฎ (ซ่อน/ลบ), สรุปจำนวนประกาศ/ผู้ใช้/รายการขายแล้ว
